# 🦖⚽ Pega Carlos!

Jogo arcade de pega-pega para celular, feito em HTML5 + JavaScript (Canvas 2D).
Carlos é o pegador: corra pela quadra de futebol society e toque em todos os
Emersons antes do tempo acabar. Na primeira captura, Carlos **vira monstro** e grita:

> "Corre mais, está correndo pouco, vaiii!!"

## 🎮 Como jogar
- 🕹️ **Joystick** (lado esquerdo): mover o Carlos
- ⚡ **Sprint**: corrida rápida (gasta fôlego)
- 😱 **Grito**: liberado após virar monstro — deixa os Emersons próximos tontos
- 🎯 Toque nos Emersons para capturá-los; capturas seguidas geram **combo**
- 🏆 Pegue todos antes do tempo acabar para vencer

## 📦 Tecnologia
- HTML5 + Canvas 2D, sem dependências
- Áudio via Web Audio API + voz via SpeechSynthesis
- Roda direto no navegador (Android/iPhone), sem instalação

## ▶️ Jogar
Abra `index.html` em um servidor (ou acesse a versão publicada no GitHub Pages).
